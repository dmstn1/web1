HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. "an HTML file"

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browsk. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
